# GLBPersonalHero-skill

你也可以直接复制这句话，发送给你的 Codex、OpenClaw 或其他 agent 助手：

```text
帮我从这个地址安装这个 skill：https://github.com/zhaocaiji/GLBPersonalHero-skill
```

This skill turns a compressed `.glb` model into a runtime Three.js personal website hero. It keeps embedded GLB animations, supports optional screen video textures, and includes four personal website templates.
